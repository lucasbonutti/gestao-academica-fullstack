# Sistema de Monitoria Acadêmica

Sistema web para gestão de monitorias acadêmicas, permitindo o controle de instituições, escolas, cursos, disciplinas, professores, alunos, monitorias e relatórios.

## Tecnologias utilizadas

### Front-end
- Angular
- TypeScript
- HTML
- CSS

### Back-end
- Java
- Spring Boot
- Spring Data JPA
- Spring Security
- JWT

### Banco de dados
- PostgreSQL

---

## Funcionalidades

- Login com autenticação
- Cadastro e gerenciamento de IES
- Cadastro e gerenciamento de escolas
- Cadastro e gerenciamento de cursos
- Cadastro e gerenciamento de disciplinas
- Cadastro e gerenciamento de professores
- Cadastro e gerenciamento de alunos
- Cadastro e gerenciamento de monitorias
- Finalização de monitorias
- Emissão de relatórios de monitoria
- Planejador visual de matriz ofertada

---

## Estrutura do projeto

### Front-end
O front-end está localizado no projeto Angular e utiliza componentes standalone, com organização por funcionalidades.

### Back-end
O back-end foi desenvolvido em Spring Boot, responsável pelas regras de negócio, persistência e autenticação.
